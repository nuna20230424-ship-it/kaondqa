# 법인카드 월폴더 자동생성 (백업)

매월 1일 다음 작업을 자동 수행하는 PowerShell 스크립트 백업본. 실제 동작은 로컬 PC의 Windows 작업 스케줄러(`법인카드 월폴더 자동생성`, 매월 1일 09:00)에서 이뤄진다.

- `monthly_folder.ps1` — 실행월 폴더 `YY.MM` + `영수증` 폴더 + 전월 양식 `법인카드 등 정산서_{전월YY}년 {전월M}월_개발QA파트.xlsx` 생성(멱등). 매개변수 `-RefDate`, `-BaseDir`, `-Master`.
- `install_task.ps1` — 위 스크립트를 작업 스케줄러에 매월 1일 09:00 실행으로 등록(StartWhenAvailable 포함).
- `manual_folder.ps1` — **수동 생성**용 래퍼. `monthly_folder.ps1`을 그대로 호출하되 원하는 월을 지정할 수 있다(예: 다음 달 미리 생성). 실행 시 월을 물어보고 엔터를 치면 이번 달. `-Month '26.08'` 처럼 인자로도 가능. 로직·결과는 자동본과 동일(멱등).
- `월폴더_수동생성.bat` — `manual_folder.ps1`을 더블클릭으로 실행하는 배치. 인자 그대로 전달(`월폴더_수동생성.bat 26.09`).

## 주의
- 스크립트 안에 로컬 절대경로(`D:\...\003. 법인카드\...`)가 박혀 있어 **백업·기록용**이다. 다른 PC에서 쓰려면 경로를 수정해야 한다.
- PowerShell 5.1은 `.ps1`을 **UTF-8 BOM**으로 저장해야 한글이 깨지지 않는다(BOM 없으면 파싱 오류).
- 복사 원본 마스터 양식: `D:\...\003. 법인카드\_양식\법인카드 등 정산서_양식.xlsx`
