# 법인카드 지출결의서 자동 작성

법인카드 영수증을 판독해 `법인카드 등 정산서` 양식을 자동으로 채우고, 영수증 파일명을 정리하는 스킬이다. 필요하면 매월 정산 폴더 자동생성까지 설정한다.

## 먼저 사용자에게 받을 것

추측하지 말고 없으면 물어볼 것.

1. **영수증 폴더 경로** — 이미지(.jpg/.png/.heic)·PDF가 든 디렉토리
2. **양식 파일 경로** — `법인카드 등 정산서` 엑셀(.xlsx)
3. **파일명 접두사** — 예: `개발QA파트_업무활동비_` (파트·명목에 따라 다름)
4. **대상 월** — 예: 2026년 5월 (영수증에서 실제 사용월을 판독하므로 폴더명과 다를 수 있음)
5. **출력 경로** — 기본 `{양식파일명}_작성본.xlsx`

## 작업 1 — 영수증 파일명 정리

각 영수증을 Read로 직접 열어 **사용일자만** 추출하고 파일명을 바꾼다.

- 규칙 = `{접두사}{YYYYMMDD}{확장자}` (예: `개발QA파트_업무활동비_20260512.jpg`)
- 같은 날짜는 시간순으로 `_2`, `_3` 부여
- 날짜 판독 실패 파일은 변경하지 말고 목록으로 보고
- **변경 전 미리보기표(원본→새이름)로 확인을 받고** 적용. 되돌리기용 매핑 로그를 남길 것
- rename 은 임시명 경유(2단계)로 순환 충돌을 피한다

## 작업 2 — 지출결의서 양식 작성

### 양식 구조 (`법인카드 등 정산서`)
- 헤더 5행 / 데이터 6~26행(최대 21건) / 합계 27행에 `=SUM(…6:…26)` 수식 존재(보존)
- B열="법인카드", C열=카드번호 뒤4자리가 미리 입력돼 있으면 덮어쓰지 말 것

### 컬럼 매핑
| 열 | 헤더 | 채울 값 |
|---|---|---|
| D | 사용(승인)일 | 영수증 사용일자 (datetime.date) |
| E | 가맹점 | 가맹점명 |
| F | 공급가 | 공급가액 (정수) |
| G | 부가세 | 부가세 (정수) |
| H | 사용(승인)액(KRW) | 합계 (정수, = F+G) |
| I | 첨부 증빙#(파일명) | 정리된 영수증 파일명 |
| J | 사용 목적 | 적요 `{가맹점} - {용도}` |
| K | 비용 계정 | 계정과목 |
| L~R | 참석인원·참석자·접대대상·김영란법·인당접대액 | **접대비일 때만** |
| S | 비고 | `[확인필요]` 등 플래그 |

### 금액 판독 규칙
- 부가세 별도 표기가 없으면 합계의 1/11을 부가세로 계산(일반과세자)
- 마트 등 혼합과세는 공급가 = 합계 − 표기 부가세
- 면세 품목은 부가세 0

### 계정과목 분류 (예시 규칙, 회사 기준에 맞게 조정)
- 식당·카페·마트(식품) → 복리후생비 (외부 접대면 접대비)
- 다이소·문구·사무용품 → 사무용품비
- 택시·교통·KTX → 여비교통비
- 주유 → 차량유지비
- 모호하면 추측하지 말고 비고(S)에 `[확인필요]` 표기 후 보고

### 기입 시 주의
- openpyxl 사용. 날짜는 진짜 날짜값, 금액은 정수
- 기존 서식·병합셀·합계 수식·선입력값(B/C)을 망가뜨리지 말 것
- 21건 초과 시 사용자에게 알리고 협의

## 작업 3 — 검증 및 보고

저장 후 보고한다.
- 처리 건수 / 전체 파일 수
- 계정과목별 건수·공급가·부가세·합계
- 월 총 지출 (공급가 + 부가세 = 합계)
- `[확인필요]` 항목 목록
- 출력 파일 경로

## (선택) 작업 4 — 매월 정산 폴더 자동생성

`scripts/` 에 동봉된 PowerShell 스크립트로 매월 1일 정산 폴더를 자동 생성한다.

- `monthly_folder.ps1` — 실행월 폴더 `YY.MM` + `영수증` 폴더 + 전월 양식 `법인카드 등 정산서_{전월}_개발QA파트.xlsx` 생성(멱등). 매개변수 `-RefDate`, `-BaseDir`, `-Master`
- `install_task.ps1` — 위 스크립트를 Windows 작업 스케줄러에 매월 1일 09:00 실행으로 등록

설정 방법
1. 두 스크립트의 `BaseDir`, `Master`, 양식 파일명 접두사를 사용자 환경에 맞게 수정
2. 마스터 양식(빈 양식)을 `Master` 경로에 둔다
3. `install_task.ps1` 실행으로 작업 등록

주의
- PowerShell 5.1은 `.ps1`을 **UTF-8 BOM**으로 저장해야 한글이 깨지지 않는다. BOM 없으면 파싱 오류가 난다
- 정산 폴더명(YY.MM)은 처리월이고 실제 사용일은 영수증에서 판독한다(전월일 수 있음)
